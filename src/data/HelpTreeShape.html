Tree Shape Help


To use Tree Shape, just open a tree file in newick format via the File menu or the toolbar button . The table next displays the number of lineages' birth date rankings consistent with the tree (shape), the probability of the tree, conditioned on its number of leaves, the total number of trees with this number of leaves and the uniform probability of a tree among them.
